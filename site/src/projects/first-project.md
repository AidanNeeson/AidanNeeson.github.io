### Description

As the title suggests, this is a game of Tic Tac Toe. It is a fairly simple game that is played by two people on one device. Challenge a friend and try to get three in a row!

### Content

The entire game was built purely out of HTML, CSS, and JavaScript in an attempt to be a very simple imlpementation of Tic Tac Toe. The source code for this prohect can be found at the project's [repository](https://github.com/AidanNeeson/TicTacToe).
